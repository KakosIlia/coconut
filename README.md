Данный проект нельзя интегрировать, загрузите исходные файлы и запустите при помощи love2D

Ваш код должен находиться в scenes/name.lua, создание первой сцены и вызов coconut прописан в main.lua
Документации для API пока что нет, все методы графических объектов лежат в файлах display.lua,physics.lua,sceneManager.lua и т.д.

Проект разрабатывается чуть больше 48 часов(информация на 16 марта 2026) так что баги, недоработки и прочие излишки присутствуют
