Данный проект нельзя интегрировать, загрузите исходные файлы и запустите при помощи love2D

Ваш код должен находиться в scenes/name.lua, создание первой сцены и вызов coconut прописан в main.lua
Временная документация: https://coconutdoc.tilda.ws/coconut/docver1

Проект разрабатывается не так долго так что баги, недоработки и прочие излишки присутствуют
